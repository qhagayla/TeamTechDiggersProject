# TeamTechDiggersProject

#### Clone this repository

```
git clone https://github.com/qhagayla/TeamTechDiggersProject.git
```
### Frontend
#### Install dependencies
```
npm install -g expo-cli
```

```
npm install
```

### Usage

#### Run the app

```
npm expo start
```

### Backend
#### Run the server
```
node loginserver.js
```
```
node signupserver.js
```
```
node meserver.js
```
```
python songlist.py
```
```
python songs.py
```
